配置文件帮助:
第一次打开该程序会判断配置文件是否存在,如果不存在则使用初始配置创建该INI配置文件
配置文件包含两部分:
[server]
host = xxx.xxx.xxx.xxx   ;数据库IP
db = MJSQL3.0   ;数据库名中联默认MJSQL3.0   共济默认historyver1 栅格默认sgdatabase
passwd = ABCabc123  ;数据库密码.默认共济的是xbrother栅格的是mysql中联的需要后台设置
port = 14333  ;数据库端口默认共济的mysql是3307,栅格的是3306，中联用的sqlserver是1433 
user = sa  ;数据库用户名默认共济的是gj,栅格的是mysql，中联的需要进sqlserver后台设置

[sheet_day]
sjl = A    ;时间所在列
gzh = 6    ;规则所在行
cj = 1    ;0选择共济  1选择中联  2选择栅格
mode =0      ；0选择按时间  1选择最后填充
filename = E:/Seafile/snmp/共济自定义报表程序/ss.xlsx  ;报表文件路径名字

报表文件格式:

需要在原来的报表文件里面添加2行,一样命名规则,一行命名测点

测点行格式:
测点行需要在规则行上一行,测点行格式A=测点ID,多个测点需要换行
中联的测点格式为:A=设备ID|测点索引,比如电表的ID为1234,电量测点在地12个测点,那个格式为:A=1234|12
共济的测点格式为:A=测点ID,测点ID=v6ID+设备ID+测点ID,那个格式为:A=E1S12A23

规则行格式:
规则行格式需要加上标示符号,目前支持2个标识DATABASE|和TEMPLATE| 
DATABASE|标识:
比如:DATABASE|JLJ(A)+JAVG(B)-JSUM(C)  这个代表用测点A的累计值 加上 测点B的平均值,减去测点C的和,A和B的来源于数据库
这里可以使用公式
JLJ这个函数会把数据库里面查询到的数据按照时间排序,然后最后一个值减去第一格式
JMAX这个函数会把数据库里面查询到的数据的最大值返回
JMIN这个函数会把数据库里面查询到的数据的最小值返回
JNEW这个函数会把数据库里面查询到的数据的最新的值返回
JOLD这个函数会把数据库里面查询到的数据的最早的值返回
JAVG这个函数会把数据库里面查询到的数据的平均值返回
JSUM这个函数会把数据库里面查询到的数据的所有数相加返回
TEMPLATE|标识:
TEMPLATE|A{}+B{}   如果是这个格式,会把TEMPLATE|后面的内容直接填入表格{}的值将会被行号替代
